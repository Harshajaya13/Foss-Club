# 🤝 How to Contribute to FOSS Club

We do not aim to compile a dry encyclopedia of Linux commands. Our goal is to maintain a **warm, oriented, and story-driven curriculum** that helps beginners understand the Linux ecosystem without feeling overloaded. 

If this repository helped you understand Linux, **pass the torch** by helping us improve it!

---

## 🛠️ How You Can Help

### 1. Found a typo, bug, or mistake?
*   Open a GitHub **Issue** describing the problem, or submit a quick **Pull Request** fixing the text directly.

### 2. Want to suggest a new tool?
To prevent this repository from turning into a bloated "Awesome Tools" list, every tool guide must focus on **problems before tools**. If you want to submit a Pull Request adding a tool, make sure it structures around:
*   ❓ **The Problem:** What roadblock or bottleneck does a beginner face?
*   🚀 **The Solution:** Why was this specific tool created to solve it?
*   💻 **The Start Command:** A single, clear startup instruction to run it immediately.

### 3. Want to run a workshop?
*   Fork this repository, customize the guides for your student audience, run your workshop, and share your experiences and improvements back with us!
